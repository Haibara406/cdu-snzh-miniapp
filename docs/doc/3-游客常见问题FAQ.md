# 游客常见问题FAQ

## 购票相关

### Q1: 门票价格是多少？

**A**: 
根据您的身份，可选择以下票型：

**普通游客：**
- 成人票：¥100/人
- 学生票：¥50/人（需出示有效学生证）
- 儿童票：¥50/人（1.2米-1.5米儿童）
- 老年票：¥50/人（60岁以上，需出示身份证）
- 家庭套票：¥220/2大1小1老（更优惠！）
- 1.2米以下儿童免票

**本地居民专享：**
- 本地人票：¥30/人（需出示本地身份证）
- 本地家庭套票：¥100/2大1小1老（需全员为本地居民）

💡 **购票提示**：
- 支持微信支付
- 购票后订单30分钟内完成支付，超时自动取消
- 电子门票，无需换票，扫码入园

### Q2: 门票包含哪些内容？

**A**: 

**包含**：
- 景区大门票（可游览所有20个景点）
- 景点介绍语音导览
- AI智能客服"竹小智"服务

**不包含**：
- 观光车票（¥30/人，建议购买）
- 索道（单程¥40，往返¥70）
- 餐饮、住宿
- 其他个人消费

### Q3: 门票有效期多久？

**A**: 
- 门票在您选择的游玩日期当天有效
- 当日可无限次出入园（请保留电子票码）
- 如需改期，请至少提前2天申请退款后重新购票

### Q4: 可以提前多久购票？

**A**: 
- 可提前30天购票
- 建议至少提前1天购买，避免当日票紧张
- 节假日建议提前3-7天购票

### Q5: 购票后如何使用？

**A**: 
1. 在小程序【我的订单】中查看订单详情
2. 点击【查看电子票】获取二维码
3. 到达景区检票口，出示二维码
4. 工作人员扫码核销后即可入园

💡 **温馨提示**：
- 一码一人，请勿截图传播
- 建议提前截图保存电子票码，以防景区内信号不佳
- 门票核销后失效，无法重复使用

### Q6: 如何退票？什么情况下可以退票？

**A**:
**退款规则**：
- ✅ 游玩日期前2天（含）以上：可全额退款
- ❌ 游玩日期前1天及当天：不支持退款
- ❌ 已核销的门票：不支持退款

**退款流程**：
1. 进入小程序【我的订单】
2. 找到对应订单，点击【申请退款】
3. 选择退款原因并提交
4. 系统自动处理，1-3个工作日原路退回

详见《门票退款政策》文档。

## 最佳游览时间

### Q24: 什么时候去最好？

**A**: 
蜀南竹海四季皆宜，各有特色：

- **春季（3-5月）**：竹笋破土，新绿满山，生机勃勃
- **夏季（6-8月）**：避暑胜地，清凉舒适，瀑飞泉涌
- **秋季（9-11月）**：秋高气爽，色彩斑斓，红叶点点
- **冬季（12-2月）**：游客较少，宁静祥和，青枝白雪

💡 可随时咨询AI客服"竹小智"查询实时天气，获取最佳游览建议。
