# 游览注意事项

## 安全须知

### 1. 森林防火

- ⚠️ **严禁吸烟、使用明火**
- ⚠️ 不得携带易燃易爆物品入园
- ⚠️ 违者将被处以罚款，情节严重者追究法律责任

竹林属于易燃物，一旦发生火灾后果不堪设想。请务必遵守防火规定，保护这片珍贵的竹海。

### 2. 游览安全

- 请走规定游览路线，不要进入未开发区域
- 注意脚下，部分路段有台阶和坡道
- 雨天路滑，请慢行并使用扶手
- 不要攀爬护栏和景观设施
- 使用观光车、索道时请遵守安全规定
- 在栈道、悬崖边等危险地带请保持距离，注意安全

### 3. 儿童安全

- 儿童需有成人陪同
- 看管好孩子，防止走失
- 教育孩子不要破坏竹林
- 不要让儿童独自靠近湖泊、悬崖等危险区域
- 儿童乘坐观光车、索道时请由成人陪同

### 4. 野生动物

- 景区内可能有蛇类等野生动物
- 请不要主动靠近或挑逗野生动物
- 发现异常及时联系工作人员
- 不要投喂野生动物

## 文明游览

### 1. 保护环境

- 🌿 爱护竹林，不要折断竹子或刻字
- 🌿 垃圾请投入垃圾桶，分类投放
- 🌿 不要采摘花草、竹笋
- 🌿 保持景区清洁卫生
- 🌿 不要在景区内乱涂乱画
- 🌿 爱护景区设施和标识

### 2. 尊重他人

- 不要大声喧哗，保持安静
- 拍照时注意不要影响他人
- 排队时请保持秩序
- 尊重工作人员的管理
- 文明使用公共设施

### 3. 爱护设施

- 不要破坏景区标识和设施
- 正确使用公共卫生间
- 爱护观光车和其他交通工具
- 不要损坏栏杆、座椅等公共设施

## 游览建议

### 1. 穿着装备

- 👟 穿舒适的运动鞋或登山鞋（避免穿高跟鞋、拖鞋）
- 👒 携带遮阳帽、太阳镜
- 🧴 涂抹防晒霜
- 🧥 根据天气携带外套（山区温差大，早晚较凉）
- 🌂 携带雨具（山区天气多变）

### 2. 物品准备

- 💧 携带足够的饮用水（建议每人1-2瓶）
- 🍫 适量零食补充体力
- 💊 常备药品（感冒药、创可贴、防蚊虫药等）
- 🔋 充电宝（山区信号可能较弱，手机耗电快）
- 📱 提前截图保存电子票码（以防信号不佳）
- 🎒 轻便背包（方便携带物品）

### 3. 拍照提示

- 📷 注意脚下安全，不要边走边拍
- 📷 使用三脚架需征得管理方同意
- 📷 禁止使用无人机（安全原因，未经许可）
- 📷 尊重他人肖像权
- 📷 最佳拍照时间：
  - 翡翠长廊：上午9:00-10:00
  - 七彩飞瀑：上午10:00-11:00，下午14:00-15:00
  - 拜寿台：清晨6:00-7:00
  - 观海楼：傍晚17:00-18:00

### 4. 身体准备

- 💪 评估自己的体力，量力而行
- 💪 高血压、心脏病患者请慎重选择爬山项目
- 💪 适当休息，不要过度疲劳
- 💪 如感不适，立即就近休息或就医
- 💪 老年人、儿童请根据自身情况选择合适的景点

## 天气提示

### ☀️ 晴天

- 注意防晒，多补水
- 上午10:00前和下午4:00后游览较舒适
- 携带遮阳帽、太阳镜、防晒霜
- 推荐景点：七彩飞瀑（彩虹效果最佳）、拜寿台、望龙坪

### 🌧️ 雨天

- 携带雨具，穿防滑鞋
- 注意路面湿滑
- 避免在雷雨天登高或靠近大树
- 暴雨天气景区可能临时关闭，请关注官方通知
- 推荐景点：蜀南竹海博物馆（室内）、翡翠长廊、仙女湖

### 🌫️ 雾天

- 能见度低，请慢行
- 跟随导游或团队，不要单独行动
- 雾中竹海别有意境，适合拍照
- 推荐景点：忘忧谷、翡翠长廊、拜寿台

### 🌡️ 炎热天气（≥32℃）

- 多补水，避开中午12:00-14:00最热时段
- 注意防暑降温，携带防暑药品
- 推荐景点：忘忧谷（竹林荫蔽）、仙寓洞（洞穴清凉）、青龙湖（水边凉爽）

### ❄️ 寒冷天气（≤5℃）

- 做好保暖，携带外套
- 早晨和傍晚温差大，注意增减衣物
- 推荐景点：蜀南竹海博物馆、天皇寺、龙吟寺

💡 **实时天气查询**：咨询AI客服"竹小智"，获取最新天气预报和游览建议。

## 特殊人群提示

### 老年人

- 选择平缓的景点（翡翠长廊、仙女湖、青龙湖、博物馆等）
- 避免需要登高的景点（天宝寨、龙吟寺、望龙坪等）
- 建议购买观光车票，减少体力消耗
- 安排充足的休息时间
- 携带常用药品
- 使用AI客服"竹小智"获取适合老年人的路线推荐

### 儿童

- 选择趣味性强的景点（忘忧谷、七彩飞瀑、湖泊类景点等）
- 避免攀爬类景点（天宝寨、竹尖漫步等）
- 每个景点30-60分钟，避免过长
- 每1.5小时安排休息
- 携带零食、水、创可贴、湿巾
- 夏季携带防蚊虫药品

### 孕妇

- 选择平缓、轻松的景点
- 避免长时间行走和爬山
- 建议购买观光车票
- 充分休息，不要过度疲劳
- 咨询AI客服"竹小智"获取适合的路线

### 行动不便者

- 景区提供轮椅免费借用服务（游客中心）
- 选择平地湖泊类景点
- 建议有陪同人员
