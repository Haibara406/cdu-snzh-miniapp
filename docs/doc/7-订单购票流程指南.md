# 订单购票流程指南

## 购票流程总览

蜀南竹海景区支持在线购票，整个流程简单便捷：

```
选择票型 → 填写信息 → 提交订单 → 完成支付 → 获取电子票 → 扫码入园
```

**预计用时**：5-10分钟

## 详细购票步骤

### 第一步：选择票型

在小程序首页或【购票】页面，选择您需要的票型：

#### 票型说明

**普通游客**：
- **成人票**：¥100/人
  - 适用：18-59周岁普通游客
  - 无需证明材料

- **学生票**：¥50/人
  - 适用：全日制在校学生
  - 入园时需出示有效学生证或学信网信息

- **儿童票**：¥50/人
  - 适用：身高1.2米-1.5米的儿童
  - 入园时需现场测量身高

- **老年票**：¥50/人
  - 适用：60周岁及以上老年人
  - 入园时需出示身份证

- **家庭套票**：¥220/套（2大1小1老）
  - 包含：2张成人票 + 1张儿童票 + 1张老年票
  - 更优惠，比单买省¥30
  - 不可拆分使用

- **1.2米以下儿童**：免票
  - 需由成人陪同入园

**本地居民专享**：
- **本地人票**：¥30/人
  - 适用：景区所在地居民
  - 入园时需出示本地身份证原件

- **本地家庭套票**：¥100/套（2大1小1老）
  - 适用：全家为本地居民
  - 入园时需全员出示本地身份证

#### 票型选择建议

- **单人出游**：选择对应的单票（成人/学生/老年）
- **家庭出游**：选择家庭套票，更优惠
- **团队出游**：多人可分别购票或联系景区咨询团队票
- **本地居民**：凭本地身份证享受专属优惠

### 第二步：填写信息

#### 必填信息

1. **游玩日期**
   - 选择您计划游玩的日期
   - 可提前30天购票
   - 建议至少提前1天购票

2. **购票数量**
   - 选择购买张数
   - 家庭套票固定为1套（含4张票）

3. **联系手机号**
   - 用于接收订单信息和电子票码
   - 确保手机号正确，以免影响入园

4. **游客信息**（可选）
   - 姓名
   - 身份证号
   - 部分优惠票型需填写

#### 注意事项

- 请确保信息准确无误
- 手机号用于接收电子票码，请仔细核对
- 优惠票型（学生票、老年票等）入园时需出示相应证件

### 第三步：提交订单

确认信息无误后，点击【提交订单】按钮。

系统会自动生成订单，包括：
- 订单号（唯一标识）
- 订单金额
- 游玩日期
- 票型信息

**⏰ 重要提示**：
- 订单创建后30分钟内必须完成支付
- 超时订单自动取消
- 系统会在订单即将过期时发送提醒（如已授权订阅消息）

### 第四步：完成支付

#### 支付方式

目前支持：
- 💳 **微信支付**

#### 支付流程

1. 点击【立即支付】按钮
2. 跳转到微信支付页面
3. 确认支付金额
4. 输入支付密码或使用指纹/面容识别
5. 完成支付

#### 支付成功

- ✅ 页面提示"支付成功"
- 📱 收到订阅消息通知（需提前授权）
- 📧 订单状态更新为"已支付"
- 🎫 自动生成电子票码

#### 支付失败

如遇支付失败：
1. 检查网络连接
2. 确认微信余额或绑定银行卡余额充足
3. 重新尝试支付
4. 如仍失败，联系AI客服"竹小智"

### 第五步：获取电子票

支付成功后，电子票码会自动生成。

#### 查看电子票

**方式一：订单详情**
1. 进入小程序
2. 点击【我的订单】
3. 找到对应订单
4. 点击【查看电子票】

**方式二：订阅消息**
- 支付成功后会收到订阅消息（需提前授权）
- 点击消息直接进入订单详情

#### 电子票信息

电子票包含：
- 二维码（用于入园核销）
- 订单号
- 票型
- 游玩日期
- 购票数量
- 购票人信息

#### 保存电子票

**建议操作**：
- 📸 提前截图保存
- ☁️ 电子票已同步到云端，可随时查看
- 📱 确保手机电量充足

**为什么要截图？**
- 景区部分区域信号可能较弱
- 避免网络问题影响入园
- 方便快速出示

### 第六步：扫码入园

#### 入园流程

1. 到达景区检票口（西门/东门/北门）
2. 出示电子票码（二维码）
3. 工作人员扫码核销
4. 核销成功，即可入园

#### 注意事项

- 🎫 **一码一人**：每张票对应一个二维码，不可重复使用
- ⏰ **当日有效**：电子票在游玩日期当天有效
- 🔄 **可多次出入**：当日凭票可无限次出入园（请保留电子票码）
- 📱 **核销后失效**：每张票核销后即失效，无法再次使用
- 👥 **多人出游**：每人需单独出示各自的电子票码

#### 优惠票型验证

**学生票**：
- 出示有效学生证或学信网信息
- 工作人员核验后放行

**儿童票**：
- 现场测量身高
- 1.2米-1.5米可使用儿童票
- 超过1.5米需补成人票差价

**老年票**：
- 出示身份证
- 核验年龄后放行

**本地人票**：
- 出示本地身份证原件
- 核验户籍地后放行

#### 入园时间

**旺季（3月-11月）**：08:00-18:00  
**淡季（12月-次年2月）**：08:30-17:30

建议早上8:00-9:00入园，人少景美，空气清新。

## 订单管理

### 查看订单

**路径**：小程序首页 → 我的订单

**订单状态**：
- **待支付**：订单已创建，等待支付
- **已支付**：支付成功，等待游玩
- **已取消**：订单已取消（超时或主动取消）
- **已退款**：已申请退款并处理完成
- **已完成**：已游玩并核销

### 申请退款

#### 退款条件

✅ **可以退款**：
- 游玩日期前2天（含）以上
- 已支付但未核销的门票

❌ **不予退款**：
- 游玩日期前1天及当天
- 已核销的门票
- 过期门票

#### 退款流程

1. 进入小程序【我的订单】
2. 找到对应订单
3. 点击【申请退款】
4. 选择退款原因：
   - 行程变更
   - 天气原因
   - 健康原因
   - 其他原因
5. 填写退款说明（可选）
6. 提交退款申请
7. 系统自动处理
8. 退款金额原路退回（1-3个工作日）

#### 退款成功

- ✅ 系统即时处理
- 💰 退款金额原路退回微信支付账户
- 📱 收到退款成功通知（如已授权订阅消息）
- 🔄 订单状态更新为"已退款"

#### 注意事项

- 退款申请一经系统处理，不可撤销
- 家庭套票不可拆分退款，必须整单退款
- 套票、优惠票退款按实际支付金额计算

详见《门票退款政策》文档。

### 订单过期

#### 过期规则

- 订单创建后30分钟内未支付，自动取消
- 系统会在订单即将过期时发送提醒（15分钟前）

#### 过期后

- 订单状态更新为"已取消"
- 不占用门票名额
- 可重新下单购票

## 购票常见问题

### Q1: 可以提前多久购票？

**A**: 可提前30天购票。建议至少提前1天购买，节假日建议提前3-7天。

### Q2: 订单支付后可以改期吗？

**A**: 不支持改期。如需改期，请至少提前2天申请退款，然后重新购票。

### Q3: 家庭套票可以分开使用吗？

**A**: 不可以。家庭套票必须同时使用，不支持拆分。

### Q4: 购票时填错信息怎么办？

**A**: 
- 如果订单未支付，可以取消订单重新下单
- 如果订单已支付但未核销，可以联系AI客服"竹小智"协助处理
- 如果已核销，无法修改

### Q5: 电子票丢失了怎么办？

**A**: 电子票已同步到云端，您可以：
1. 进入小程序【我的订单】
2. 找到对应订单
3. 重新查看电子票码

无需担心丢失！

### Q6: 可以帮别人购票吗？

**A**: 可以。购票时填写游玩人的信息即可。但建议让游玩人自己保存电子票码，方便入园。

### Q7: 景区现场可以购票吗？

**A**: 可以，但建议提前在线购票：
- 避免排队等候
- 享受在线优惠
- 提前规划行程
- 节假日现场可能售罄

### Q8: 购票后多久能收到电子票？

**A**: 支付成功后即时生成电子票，可在【我的订单】中查看。

### Q9: 儿童身高在临界点怎么办？

**A**: 入园时现场测量身高：
- 1.2米以下：免票（需成人陪同）
- 1.2米-1.5米：儿童票¥50
- 1.5米以上：成人票¥100

如果购买了儿童票但现场测量超高，需补差价。

### Q10: 学生证忘带了怎么办？

**A**: 
- 可以提供学信网电子证明
- 如无法提供，需补成人票差价
- 建议购票前确认是否能提供学生证明
