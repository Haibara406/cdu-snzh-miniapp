# 蜀南竹海智能路线推荐指南

> **⚠️ 重要提示（仅供AI参考）**  
> 本文档为AI路线推荐的执行指南，包含详细的推荐策略和示例对话。
> **示例对话仅表达大致结构和流程，具体内容和格式可能随系统更新而变化，不具备时效性。**
> AI应根据实际情况灵活调整，以实际工具返回的数据和系统提示词为准。

## 路线推荐原则

### 1. 天气适配原则

根据当天或游玩日的天气情况，智能调整推荐景点：

#### 晴天路线

- **优先推荐**：七彩飞瀑（彩虹效果最佳）、拜寿台（云海观景）、望龙坪（远眺全景）
- **次要推荐**：观海楼、青云长廊、竹尖漫步
- **注意事项**：提醒游客做好防晒，携带遮阳帽、太阳镜和防晒霜

#### 阴天路线

- **优先推荐**：忘忧谷（竹林光影效果佳）、翡翠长廊（漫射光线柔和）、花溪十三桥
- **次要推荐**：天宝寨、仙女湖、海中海
- **注意事项**：云雾天气竹海景色独特，是摄影的好时机

#### 雨天路线

- **优先推荐**：蜀南竹海博物馆（室内）、翡翠长廊（有遮蔽）、仙女湖（雨中湖景）
- **避免推荐**：天宝寨（登高危险）、七彩飞瀑（路滑）、竹尖漫步（栈道湿滑）
- **注意事项**：务必携带雨具，穿防滑鞋，注意安全

#### 炎热天气（≥32℃）

- **优先推荐**：忘忧谷（竹林荫蔽凉爽）、仙寓洞（洞穴清凉）、青龙湖（水边凉爽）、七彩飞瀑（飞泉凉爽）
- **次要推荐**：翡翠长廊、花溪十三桥（溪流边）
- **避免推荐**：需要长时间暴晒的开阔景点
- **注意事项**：多补水，避开中午12:00-14:00最热时段

#### 寒冷天气（≤5℃）

- **优先推荐**：蜀南竹海博物馆、天皇寺、龙吟寺（活动量适中）
- **注意事项**：做好保暖，早晨和傍晚温差大

### 2. 人群适配原则

#### 亲子游（有小孩）

**适合景点**：
- 忘忧谷（难度低，趣味性强）
- 七彩飞瀑（视觉冲击大，小孩喜欢）
- 仙女湖、青龙湖、海中海（湖泊景观，可游船）
- 蜀南竹海博物馆（科普教育）
- 翡翠长廊（轻松漫步）

**不适合景点**：
- 天宝寨（需要攀爬，小孩体力不支）
- 竹尖漫步（栈道较高，有安全隐患）
- 龙吟寺（海拔较高，路程较长）

**特别建议**：
- 游览时间：每个景点30-60分钟，避免过长
- 休息频率：每1.5小时安排休息
- 午餐时间：12:00-13:30，避免小孩饥饿
- 携带物品：零食、水、创可贴、湿巾
- 建议购买观光车票，减少步行距离

#### 老年游（有老人）

**适合景点**：
- 翡翠长廊（平缓步道）
- 仙女湖、青龙湖（平地湖泊）
- 蜀南竹海博物馆（室内，无需爬坡）
- 七彩飞瀑（观景台平缓）
- 天皇寺（台阶较少）
- 海中海（平地湖泊）

**不适合景点**：
- 天宝寨（登高劳累）
- 竹尖漫步（体力要求较高）
- 龙吟寺（海拔较高）
- 望龙坪、拜寿台（需要登高）

**特别建议**：
- 游览节奏：慢节奏，充分休息
- 观光车：强烈建议购买观光车票
- 午休时间：13:00-14:00安排午休
- 医疗保障：提醒携带常用药品
- 紧急联系：告知景区医务室位置
- 轮椅服务：游客中心提供轮椅免费借用服务

#### 徒步爱好者

**推荐路线**：
- 竹尖漫步（1.7公里栈道）
- 忘忧谷（溪流徒步）
- 天宝寨（登高挑战）
- 花溪十三桥（1公里溪边步道）
- 青云长廊（观景徒步）

**特色体验**：
- 全程徒步穿越路线：西门 → 忘忧谷 → 翡翠长廊 → 竹尖漫步 → 仙寓洞 → 海中海 → 东门（约6-8小时）

**注意事项**：
- 穿专业徒步鞋
- 携带登山杖
- 准备充足的水和能量食品
- 注意体力分配

#### 摄影爱好者

**最佳拍摄点**：
1. **七彩飞瀑**：上午10:00-11:00，下午14:00-15:00（彩虹效果最佳）
2. **翡翠长廊**：上午9:00-10:00（光影效果）
3. **拜寿台**：清晨6:00-7:00（云海日出）
4. **忘忧谷**：全天（竹林光影）
5. **观海楼**：傍晚17:00-18:00（夕阳竹海）
6. **望龙坪**：日出、日落（远眺全景）

**摄影建议**：
- 携带广角镜头（拍竹海全景）
- 携带长焦镜头（拍细节和远景）
- 三脚架（慢门拍瀑布，需征得管理方同意）
- 偏振镜（消除反光）
- 避开游客高峰期（10:00-15:00）

#### 休闲游

**推荐景点**：
- 翡翠长廊（轻松漫步）
- 仙女湖、海中海（湖边休憩）
- 蜀南竹海博物馆（文化体验）
- 七彩飞瀑（打卡拍照）
- 天皇寺（禅意休闲）

**行程特点**：
- 慢节奏，不赶时间
- 景点间充分休息
- 享受美食（安排品尝特色竹笋宴）
- 下午茶时光（景区茶馆）

### 3. 时长适配原则

#### 半日游（4小时）

**精华路线**：
- **路线A（上午）**：08:30-09:00 入园 → 09:00-11:00 忘忧谷（2小时）→ 11:00-12:00 七彩飞瀑（1小时）→ 12:00-12:30 翡翠长廊（30分钟）
- **路线B（下午）**：14:00-14:30 入园 → 14:30-16:00 翡翠长廊+观海楼（1.5小时）→ 16:00-17:00 七彩飞瀑（1小时）→ 17:00-18:00 仙女湖（1小时）

**景点数量**：2-3个核心景点  
**节奏**：紧凑但不仓促  
**建议**：购买观光车票

#### 一日游（6-8小时）

**经典路线**：
- **上午**：08:00-08:30 入园 → 08:30-10:30 忘忧谷（2小时）→ 10:30-11:30 翡翠长廊（1小时）→ 11:30-12:00 观海楼（30分钟）
- **午餐**：12:00-13:30 景区餐厅
- **下午**：13:30-14:30 七彩飞瀑（1小时）→ 14:30-15:30 天宝寨或仙寓洞（1小时）→ 15:30-16:30 海中海或仙女湖（1小时）→ 16:30-17:00 返程

**景点数量**：5-6个景点  
**节奏**：适中，有充分游览时间  
**建议**：必买观光车票

#### 两日游（16小时）

**第一天**（西线）：
- 上午：西门入园 → 忘忧谷 → 蜀南竹海博物馆 → 翡翠长廊
- 下午：观海楼 → 竹尖漫步 → 仙寓洞 → 天宝寨
- 傍晚：观日落，入住景区酒店

**第二天**（东线）：
- 上午：七彩飞瀑 → 花溪十三桥 → 青云长廊 → 龙吟寺
- 下午：海中海 → 仙女湖 → 青龙湖 → 东门出园

**景点数量**：10-12个景点  
**节奏**：慢节奏，深度体验  
**建议**：住宿景区内，体验竹海夜景和清晨

### 4. 距离优化原则

#### 景点位置分布

**西部区域**：
- 西门
- 忘忧谷
- 蜀南竹海博物馆
- 翡翠长廊
- 观海楼
- 墨溪
- 拜寿台

**中部区域**：
- 竹尖漫步
- 仙寓洞
- 天宝寨
- 海中海
- 仙女湖

**东部区域**：
- 七彩飞瀑
- 花溪十三桥
- 青云长廊
- 龙吟寺
- 青龙湖
- 东门

#### 最短路径规划

采用"就近原则"，避免来回往返：
1. **西进东出**：西门 → 西部景点 → 中部景点 → 东部景点 → 东门
2. **东进西出**：东门 → 东部景点 → 中部景点 → 西部景点 → 西门
3. **环线游览**：适用于两日游，第一天走一半，第二天走另一半

## AI推荐策略（AI执行逻辑）

> **重要**：以下是AI执行路线推荐时的详细流程和规范。

### 问题清单（必答项）

在生成路线前，AI必须先询问以下问题，确保推荐的准确性：

#### 1. 游玩基本信息

**游玩日期**："您打算哪天来玩？"
- 用于查询当天天气，适配推荐路线
- 如果是"明天"、"后天"等相对时间，需调用天气工具
- 必须获取具体日期

**游玩时长**："计划游玩多长时间？半天、1天还是2-3天？"
- 半天（4小时）：推荐2-3个精华景点
- 一天（6-8小时）：推荐5-6个景点
- 两天以上：推荐10+景点，建议住宿

#### 2. 同行人员构成（多选）

**是否有小孩？**
- 如有：
  - 避免推荐天宝寨、竹尖漫步等攀爬景点
  - 优先推荐：忘忧谷、七彩飞瀑、仙女湖、博物馆、翡翠长廊
  - 提醒：每个景点30-60分钟，每1.5小时安排休息
  - 建议携带零食、水、创可贴、湿巾

**是否有老人？**
- 如有：
  - 避免推荐需要登高的景点（天宝寨、龙吟寺、望龙坪、拜寿台）
  - 优先推荐：翡翠长廊、仙女湖、青龙湖、博物馆、七彩飞瀑观景台
  - 强烈建议购买观光车票
  - 安排午休时间（13:00-14:00）
  - 提醒携带常用药品

**是否行动不便/需轮椅？**
- 仅推荐平地湖泊类景点
- 提醒：游客中心提供轮椅免费借用服务

#### 3. 出行方式（必答）

**自驾 / 公共交通 / 包车？**

**如果自驾：**
- 询问："是燃油车还是电动车（EV）？"
- **如果是EV**：
  - 推荐路线时标注最近的充电桩位置
  - 列出充电站名称、位置、距离
  - 提醒：景区内共有14个充电站，建议提前规划
  - 主要充电站：快鳗充电站、川能充电站、特斯拉目的地充电站
- **如果是燃油车**：
  - 推荐最近的停车场位置
  - 标注停车场名称、距离
  - 景区停车场免费，但节假日车位紧张

#### 4. 游玩偏好（可选）

**轻松休闲 / 徒步探险 / 摄影打卡？**
- **轻松休闲**：湖泊、长廊类景点，节奏慢，多安排休息
- **徒步探险**：竹尖漫步、忘忧谷、天宝寨、花溪十三桥
- **摄影打卡**：七彩飞瀑、翡翠长廊、拜寿台、忘忧谷、观海楼，标注最佳拍照时间

### 必须调用的工具

推荐路线时，AI需调用以下工具（已集成在`recommendRoute`工具中）：

1. **queryForecastWeather（查询天气）**
   - 参数：游玩日期
   - 用于：天气适配原则

2. **listAvailableTickets（查询门票）**
   - 用于：告知用户门票价格
   - 根据人群给出推荐票型

3. **listAllScenics（查询景点）**
   - 用于：获取景点详细信息

4. **queryFacilities（查询设施）**
   - 参数：设施类型（停车场、充电桩、餐厅、住宿等）
   - 用于：提供具体的设施位置信息

**注意**：AI不需要手动调用以上工具，只需调用`recommendRoute`工具，系统会自动整合所有信息。

### AI推荐输出格式

生成路线时，必须包含以下信息：

#### 1. 天气提示

```markdown
📅 **游玩日期**：{日期}
🌤️ **天气预报**：{温度}℃，{天气状况}，{风力}
👕 **穿衣建议**：{根据温度和天气给出建议}
☂️ **装备建议**：{是否需要雨具、防晒等}
```

#### 2. 路线规划

```markdown
🗺️ **推荐路线**（{人群类型}，{时长}）：

**上午**：
1. {时间段} {景点名称}（{建议游玩时长}）
   - 📍 特色：{景点特色}
   - 🚶 路线：{如何到达}
   
2. {时间段} {景点名称}（{建议游玩时长}）
   - 📍 特色：{景点特色}
   - 🚶 路线：{如何到达}

**午餐**：{时间段} {推荐餐厅}
- 📍 位置：{餐厅名称}
- 💰 人均：¥{价格区间}
- 🥘 推荐：{特色菜品}

**下午**：
{继续安排景点}

**傍晚**：{根据游玩天数给出住宿建议或返程建议}
```

#### 3. 交通与停车建议

```markdown
🚗 **自驾信息**：
- 停车场推荐：{停车场名称}（距{景点}约{距离}米）

{如果是EV}
⚡ **充电桩推荐**：
1. {充电站名称}（{位置描述}，距离约{距离}米）
2. {充电站名称}（{位置描述}，距离约{距离}米）

🚌 **景区交通**：
- 观光车：建议购买（¥30/人），停靠{数量}个站点
- 索道：{如果路线涉及索道，标注费用和位置}
```

#### 4. 温馨提示

```markdown
💡 **温馨提示**：
- {根据人群特点给出个性化建议}
- {根据天气给出装备建议}
- {根据时长给出节奏建议}
- 🎫 门票：建议提前在线购买，避免排队
- 💬 导航：有任何问题可随时咨询我，我会为您导航到目的地
```

#### 5. 设施位置标注

```markdown
📍 **重要设施位置**：
- 卫生间：{景点附近的卫生间位置}
- 医务室：{游客中心医务室位置}
- 游客中心：{西门/东门游客中心位置}
- 紧急联系：120（医疗急救）、110（报警）
```

### 默认假设规则

如果用户未回答某些问题，使用以下默认值并提示用户：

```markdown
💡 **根据您的描述，我做了以下假设**：
- 游玩时长：一日游（6-8小时）
- 同行人员：成年人
- 出行方式：自驾（燃油车）
- 游玩偏好：休闲游览

如有不符，请告诉我，我会为您调整路线哦！
```

### 禁止行为

❌ **AI不应该：**
1. 在未获取足够信息前就直接推荐路线
2. 忽略用户的特殊人群需求（老人、小孩）
3. 推荐不适合当天天气的景点
4. 给出模糊的时间安排（必须精确到时间段）
5. 忘记标注停车场或充电桩位置（如果用户自驾）
6. 推荐超出用户时长预算的路线
7. 机械地复述工具返回的内容（要用自己的语言解释）
8. 合并描述景点（每个景点必须单独介绍）

### 动态调整策略

根据用户反馈实时调整：
- 用户说"太累了"：减少景点数量，增加休息时间
- 用户说"想多看看"：增加景点，调整游玩时长
- 用户说"想换个景点"：询问原因，推荐替代景点
- 用户说"不想爬山"：调整为平缓路线
- 用户说"想看瀑布"：优先推荐七彩飞瀑

## 示例对话（仅供参考）

> **⚠️ 重要提示**：以下示例对话仅表达大致结构和流程，具体内容和格式可能随系统更新而变化，不具备时效性。AI应根据实际情况和工具返回的数据灵活调整。

### 示例1：完整信息的路线推荐

**用户**："我明天带着父母和5岁的孩子来玩一天，我们是自驾电动车，老人腿脚不太好，孩子很喜欢拍照。"

**AI**：
```
了解啦！我特意为您的家庭出游精心规划了一条路线~ 😊

📅 游玩日期：2025年10月21日
🌤️ 天气预报：晴，25℃，微风，湿度65%
👕 穿衣建议：轻薄外套+长裤，早晚稍凉
☂️ 装备建议：遮阳帽、防晒霜、太阳镜、充足饮水

考虑到您家里有老人和小朋友，我推荐以下轻松舒适的游玩路线：
```

## 温馨提示总结

### 必备物品

- ✅ 舒适的运动鞋或登山鞋
- ✅ 防晒用品（帽子、太阳镜、防晒霜）
- ✅ 饮用水（建议每人1-2瓶）
- ✅ 雨具（山区天气多变）
- ✅ 充电宝（拍照耗电）
- ✅ 常用药品（创可贴、感冒药等）

### 购票建议

- 🎫 提前在线购票，节省排队时间
- 🚌 强烈建议购买观光车票（¥30/人）
- 🚡 索道可选（单程¥40，往返¥70）

### 餐饮建议

- 🍽️ 景区内有多家餐厅，人均¥50-80
- 🥢 推荐品尝：竹笋宴、竹筒饭、腊肉
- 🍱 可自带食物，景区有休息区

### 住宿建议

- 🏨 景区内外均有酒店、民宿
- 💰 价格范围：¥200-800/晚
- 📱 建议提前预订（尤其是节假日）

### 最佳游览时间

- 🌸 **春季（3-5月）**：新笋破土，竹林新绿
- ☀️ **夏季（6-8月）**：避暑胜地，清凉舒适
- 🍂 **秋季（9-11月）**：秋高气爽，色彩斑斓
- ❄️ **冬季（12-2月）**：游客较少，宁静祥和

### 注意事项

- ⚠️ 严禁吸烟和使用明火
- ⚠️ 不要破坏竹林和采摘植物
- ⚠️ 看管好随身物品和小孩
